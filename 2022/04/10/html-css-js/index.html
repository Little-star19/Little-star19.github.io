HTML+CSS+JS基础知识

Web标准的三大组成部分

标准 说明
结构 结构用于对网页元素进行整理和分类，HTML
表现 表现用于设置网页元素的排版、颜色、大小等外观样式，CSS
行为 行为是指网页模型的定义以及交互的编写，JavaScript

  • 生成兄弟标签：+
  • 生成带有类名或id名字的：.类名/id名
  • 生成类名是有顺序的：自增符号$

    • 如lh26————line-height:26px;

    • 锚点链接：href=”#id值”，快速定位到页面中的某个位置。

CSS层叠样式表，标记语言。

  • 引入方式：
  • 1.行内样式表（行内式）:在标签内部的style属性中。
    <标签名 style="属性1：属性值1；属性2：属性值2；...；">

CSS选择器

  • 选择标签用的，分为基础选择器和复合选择器两大类。
  • 基础选择器:标签选择器、类选择器、id选择器、通配符选择器
  • 复合选择器：后代选择器、子选择器、并集选择器、伪类选择器

标签选择器

为同类型标签统一设置样式

标签名 {样式声明}

类选择器

为某一个或某几个标签设置样式，标签内增加class=”类名”。

.类名 {样式声明}
  • 多类名:在标签class属性中写多个类名
    • class=”类名1 类名2 …”

id选择器

为标有特定id的HTML元素指定特定的样式，一般和js搭配，标签内增加id=”类名”。

#类名 {样式声明}

通配符选择器

为页面所有元素设置样式。

* {样式声明}

后代选择器

选择父元素里面的子元素，通过【空格】分隔。

选择器1 选择器2 {样式声明}

子选择器

只选择作为某元素最近一级子元素,通过【>】分隔。

选择器1>选择器2 {样式声明}

并集选择器

为选择多组标签定义相同样式,通过【,】分隔

选择器1,选择器2 {样式声明}

伪类选择器

为某些选择器添加特殊的效果,通过【:】分隔
有链接伪类选择器、结构伪类选择器

链接伪类选择器

a:link {样式声明}    /*选择所有未被访问的链接*/
a:visited {样式声明} /*选择所有已被访问的链接*/
a:hover {样式声明}   /*选择鼠标指针位于脐上的链接*/
a:active {样式声明}  /*选择鼠标按下未弹起的链接*/
  • 按LVHA顺序声明
  • a链接在浏览器中有默认样式，需要给链接单独指定样式。

:focus伪类选择器

用于获取聚焦（就是光标）的表单元素
一般input类表单标签才能获取

input:focus {样式声明}

CSS的三大特性

  • 层叠性：相同选择器设置相同的样式，后面代码覆盖前面代码。
  • 继承性：子标签继承父标签的某些样式，如text-,font-,line-开头的，以及color属性
  • 优先级：！important>行内样式>id选择器>类选择器，伪类选择器>元素选择器>继承或*
  • 选择器优先级权重
选择器 权重
！important ~无穷大
行内样式 1,0,0,0
id选择器 0,1,0,0
类选择器，伪类选择器 0,0,1,0
元素选择器 0,0,0,1
继承或* 0,0,0,0
  • 复合选择器会有CSS权重叠加的问题，如：
    • div ul li————0,0,0,3
    • .nav ul li————0,0,1,2
    • a:hover—————0,0,1,1
    • .nav a——————0,0,1,1

CSS的元素显示模式

显示模式就是元素以什么方式进行显示，比如div自己占一行，一行可以放多个span
html一般分为块元素和行内元素两种类型

块元素

常见的块元素：<h1>~<h6>、<p>、<div>、<ul>、<ol>、<li>等
其中<div>是最典型的块元素
  • 独占一行
  • 高度、宽度、内外边距都可以控制
  • 宽度默认是容器（父级宽度）的100%
  • 是一个容器及盒子，里面可以放行内或块级元素
  • 文字类的元素不能使用块级元素p、h1~h63

行内元素

常见的行内元素：<a>、<strong>、<b>、<em>、<i>、<del>、<s>、<ins>、<u>、<span>等
其中<span>是典型的行内元素
  • 相邻行内元素在一行上，一行可以显示多个
  • 高、宽直接设置是无效的
  • 默认宽度就是它本身内容的宽度
  • 行内元素只能容纳文本或其他行内元素
  • 连接内不能再放链接，放块级元素，但转换块级模式最安全。

行内块元素

常见的行内块元素：<img />、<input />、<td>
  • 和相邻行内或行内块元素在一行上，但之间会有空空隙，一行可以显示多个
  • 默认宽度就是它本身内容的宽度
  • 高度、宽度、内外边距都可以控制

显示模式转换

display:block/inline/inline_block;

盒子模型

  • 盒子模型：就是把HTML页面中的布局元素看作是一个矩形的盒子，也就是装内容的容器，它包括：边框、外边距、内边距和实际内容。

边框

设置边框宽度、样式、颜色
盒子的宽度width值不包括边框

border-width:
border-style:solid实线/dashed虚线/dotted电线/...
border-color:
border-collapse:collapse;/*合并相邻边框*/
/*复合属性：没有顺序*/
border:1px solid red
border-top:
border-bottom:
border-left:
border-right:

padding

内边距也会影响盒子实际大小，没有指定width、height就不撑宽大小。

padding-left:
padding-top:
...
/*复合属性*/
padding:5px/*上下左右都是5像素*/
padding:5px 10px/*上下是5,左右是10*/
padding:5px 10px 20px/*上是5,左右是10,下是20*/
padding:5px 10px 20px 30px/*上右下左，顺时针*/

margin

基本同padding

  • 实现块级盒子水平居中:
    -必须指定宽度width
    -盒子左右外边距设为auto,常见写法margin:10px auto
  • 嵌套块元素垂直外边距的塌陷，解决方案：
    • 为父元素定义上边框
    • 为父元素定义上内边距
    • 为父元素添加overflow:hidden
  • 网页很多都带有默认的内外边距，清除内外边距:
    *{
	padding:0;
	margin:0;
}
  • 行内元素为了照顾兼容性，尽量只设置左右内外边距，不要设置上下，但转换为块级或行内块元素就可以了。

三种布局方式

  • 多个块级元素纵向排列找标准流，多个块级元素横向排列找浮动。
  • 为了约束浮动元素位置，网页布局一般采用的策略：先用标准流的父元素排列上下位置，之后内部资源素材去浮动排列左右位置。

标准流

标准流就是标签按照默认方式排列

浮点流

流动流可以改变元素标签默认的排列方式
最经典的应用：让多个块级元素一行内排列显示

float： none/left/right；
  • 浮动元素会脱离标准流（脱标）
    -脱离标准普通流的控制（浮）移动到到指定位置（动）
    -浮动的盒子不再保留原先的位置
  • 浮动的元素会一行内显示并且元素顶部对齐
  • 浮动的元素会具有行内块元素的特性
    -行内元素有了浮动，则不需要转换为块级元素或行内块元素，就可以指定宽高。
    -块级盒子没有设置宽度，默认宽度和父级一样宽，添加浮动后，大小根据内容来决定。

【清除浮动】
本质是清除浮动元素造成的影响。
如果父盒子本身有高度，则不需要清除浮动。
清除浮动后，父级就会根据浮动的子盒子自动检测高度。父级有了高度，就不会影响下面的浮动流了。

清除浮动的方法：
1.额外标签法（隔墙法）:在浮动元素末尾添加一个空标签。
/*举例*/
 <div class="clear"> </ div>
 .clear{
	clear:both/left/right; 
	}
2.父级添加overflow属性
overflow:hidden/auto/scroll;
3.父级添加after伪元素
>相当于升级版的隔墙法，不过不用额外写一个标签。 
.clearfix:after{
	content:"";
	display:block;
	height:;
	clear:both;
	visibility:hidden;
}
.clearfix{
/*为兼容IE6 7*/
 	*zoom:;
}
4.父级添加双伪元素 
.clearfix:after,.clearfix:after{
	content:"";
	display:table;
}
.clearfix:after{
	clear:both;
}
learfix{
/*为兼容IE6 7*/
 	*zoom:;
}
### 定位流 >定位：将盒子定在某一个位置。
定位=定位模式+边偏移
定位模式用于指定一个元素在文档中的定位方式。边偏移则决定了该元素的最终位置。
  • 定位模式
    position:static（静态定位）/relative（相对定位）/absolute（绝对定位）/fixed（固定定位）
  • 边偏移
    top:几px;   //顶部偏移量，定位元素相对于父元素上边线的距离。
bottom:几px;//底部偏移量，定位元素相对于父元素下边线的距离。
left:几px;  //左侧偏移量，定位元素相对于父元素左边线的距离。
right:几px; //右侧偏移量，定位元素相对于父元素右边线的距离。
